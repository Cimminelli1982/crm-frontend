Start the new CRM frontend with the correct command.

## Command

Run: `PORT=3002 npm run new-crm:dev`

## URL

After starting, the app will be available at: **http://localhost:3002/new-crm/command-center**

## Note

Do NOT use `npm start` - that starts the old CRM which redirects to `/contacts/interactions`.
